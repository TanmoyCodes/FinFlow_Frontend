# 🎨 FinFlow Frontend

A modern, high-performance financial dashboard built with **React 19** and **Tailwind CSS**. This repository contains the user and administrator interfaces for the FinFlow Microservices ecosystem.

---

## ✨ Features

- **Responsive Design**: Fully optimized for Desktop, Tablet, and Mobile.
- **Dynamic Analytics**: Interactive charts and data visualizations using Recharts.
- **Smooth Animations**: Powered by Framer Motion for a premium user experience.
- **Glassmorphism UI**: Modern aesthetic with subtle blurs and gradients.
- **State Management**: Clean and efficient state handling using Zustand.
- **Form Handling**: Integrated with `react-dropzone` for seamless file uploads.

---

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite 8](https://vitejs.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/) & [React Icons](https://react-icons.github.io/react-icons/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Charts**: [Recharts](https://recharts.org/)
- **State**: [Zustand](https://github.com/pmndrs/zustand)
- **API Client**: [Axios](https://axios-http.com/)

---

## 📂 Project Structure

```text
src/
├── components/     # Reusable UI components (Buttons, Modals, Cards)
├── pages/          # Full page views (Dashboard, Login, Admin)
├── store/          # Zustand store for global state management
├── services/       # API integration layer with Axios
├── hooks/          # Custom React hooks
├── utils/          # Helper functions and constants
└── assets/         # Images, fonts, and global styles
```

## 🚀 Getting Started

### Prerequisites
- [Node.js 20+](https://nodejs.org/)
- [npm](https://www.npmjs.com/)

### Installation

1. Clone the repository and navigate to the project directory:
   ```bash
   cd FinFlow_Frontend
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

3. Create a `.env` file in the root and add your API Gateway URL:
   ```env
   VITE_API_BASE_URL=http://localhost:8080
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

5. Build for production:
   ```bash
   npm run build
   ```
